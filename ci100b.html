How to enter RAS

  1. Pull out upper unit.
  2. Power off machine.
  3. Pull up dip switch 1 on switch pack 2..
  4. Power on machine and use buttons to run RAS commands.
Operation RAS Code
Reset Machine 0701

Operational Cover

Firstly, use the yellow dot key to unlock and remove the Collection Box. Next use the red dot key to unlock and rack out the main machine.

Step 1

Undo the 2 screws indicated and remove the metal cover.

Step 2

With the cover removed, unplug connectors K11, 12 & 13.

Step 3

The cover is secured in place by 4 screws; 1 on the left-hand side.
